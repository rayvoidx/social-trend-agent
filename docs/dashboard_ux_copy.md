## 📋 인사이트 카드

### 카드 제목

| 인사이트 유형 | 제목 패턴 | 예시 |
|--------------|----------|------|
| trend | "[키워드] 관심 급증" | "친환경 패키지 관심 급증" |
| sentiment | "[주제] 감성 변화 감지" | "브랜드A 긍정 감성 상승" |
| viral | "[콘텐츠] 바이럴 기회" | "언박싱 챌린지 바이럴 기회" |
| competitor | "[경쟁사] 동향 포착" | "경쟁사B 신제품 출시" |
| opportunity | "[카테고리] 마케팅 기회" | "Z세대 에코뷰티 기회" |

### 카드 설명

**구조**: 핵심 발견 + 수치 + 의미

**예시**:
```
최근 7일간 '#친환경패키지' 해시태그 언급이 320% 급증했습니다.
Z세대(18-24세) 비중이 78%로, 언박싱 콘텐츠 제작 기회입니다.
```

### 뱃지/태그

| 태그 | 색상 | 용도 |
|------|------|------|
| 높은 실행가능성 | 초록 | actionability: high |
| 즉시 대응 필요 | 빨강 | urgency: high |
| 긍정 트렌드 | 파랑 | positive trend |
| 주의 필요 | 주황 | negative trend |
| 새로운 기회 | 보라 | opportunity |

## 📝 상태 메시지

### 분석 진행 상태

| 상태 | 메시지 | 설명 |
|------|--------|------|
| pending | 분석 대기 중... | 작업이 대기열에 있습니다 |
| running | 데이터 수집 중... | 뉴스/소셜 데이터를 수집하고 있습니다 |
| running | 감성 분석 중... | 수집된 데이터를 분석하고 있습니다 |
| running | 인사이트 생성 중... | LLM이 인사이트를 생성하고 있습니다 |
| completed | 분석 완료 | 리포트가 준비되었습니다 |
| failed | 분석 실패 | 오류가 발생했습니다. 다시 시도해주세요 |

### 에러 메시지

| 에러 유형 | 메시지 |
|----------|--------|
| API 키 없음 | API 키가 설정되지 않았습니다. 설정 페이지에서 키를 입력해주세요. |
| 할당량 초과 | API 호출 한도에 도달했습니다. 잠시 후 다시 시도해주세요. |
| 네트워크 오류 | 서버에 연결할 수 없습니다. 인터넷 연결을 확인해주세요. |
| 데이터 없음 | 검색 결과가 없습니다. 다른 키워드로 검색해보세요. |
| 시간 초과 | 요청 시간이 초과되었습니다. 다시 시도해주세요. |

## 🏠 첫 진입 안내

### 웰컴 메시지

```
👋 AI 트렌드 분석 대시보드에 오신 것을 환영합니다!

이 대시보드에서 할 수 있는 것:
• 뉴스, 소셜 미디어, 바이럴 비디오 트렌드 분석
• 데이터 기반 마케팅 인사이트 도출
• 인사이트를 마케팅 미션으로 전환
• 최적의 크리에이터 매칭

시작하려면 "새 분석 시작" 버튼을 클릭하세요.
```

### 온보딩 단계

**Step 1: 에이전트 선택**
```
어떤 트렌드를 분석할까요?

📰 뉴스 트렌드
언론 보도 및 업계 동향 분석
→ 경쟁사 모니터링, 시장 조사에 적합

🔥 바이럴 비디오
YouTube, TikTok 급상승 콘텐츠 분석
→ 콘텐츠 기획, 인플루언서 협업에 적합

📱 소셜 트렌드
X, Instagram, 네이버 블로그 반응 분석
→ 소비자 인사이트, 브랜드 모니터링에 적합
```

**Step 2: 키워드 입력**
```
분석할 키워드를 입력하세요

💡 팁:
• 브랜드명, 제품명, 해시태그 등 구체적인 키워드
• 여러 키워드는 쉼표로 구분 (예: AI, 마케팅, 자동화)
• 한글/영문 모두 가능
```

**Step 3: 기간 설정**
```
분석 기간을 선택하세요

⏱️ 24시간: 실시간 이슈, 바이럴 초기 감지
📅 7일: 일반적인 트렌드 파악 (권장)
📆 30일: 장기 트렌드, 전략 수립
```

## 📌 툴팁 & 헬프 텍스트

### 입력 필드 헬프

| 필드 | 플레이스홀더 | 헬프 텍스트 |
|------|-------------|-------------|
| 키워드 | AI 마케팅, 브랜드명 | 분석할 주제나 브랜드를 입력하세요 |
| 기간 | 7d | 24h, 7d, 30d 중 선택 |
| 플랫폼 | youtube | youtube, tiktok, instagram 중 선택 |
| 시장 | KR | KR, US, JP 중 선택 |

### 메트릭 설명

| 메트릭 | 툴팁 |
|--------|------|
| 참여율 | 좋아요+댓글+저장 / 노출 수 × 100 |
| Z-Score | 평균 대비 급상승 정도 (2.0 이상이면 상위 2.5%) |
| 커버리지 | 분석에 사용된 데이터 양의 충분성 |
| 실행가능성 | 인사이트를 바로 액션으로 전환할 수 있는 정도 |

## 🎨 Empty State

### 데이터 없음

```
📭 아직 분석 결과가 없습니다

첫 번째 트렌드 분석을 시작해보세요.
"새 분석 시작" 버튼을 클릭하면
1분 안에 인사이트를 확인할 수 있습니다.

[새 분석 시작]
```

### 검색 결과 없음

```
🔍 검색 결과가 없습니다

"{키워드}"에 대한 결과를 찾을 수 없습니다.
다음을 시도해보세요:

• 다른 키워드로 검색
• 기간을 늘려서 검색
• 필터 조건 완화
```

### 오프라인

```
📡 인터넷 연결이 끊어졌습니다

인터넷 연결을 확인한 후 다시 시도해주세요.
연결되면 자동으로 데이터를 새로고침합니다.

[다시 시도]
```
